- present a fork, deployment, hosted service, public network, or release as the

- use the Obscura47 name, logos, or branding in a way that implies endorsement,

## Forks And Deployments

Forks and derivative deployments should use clearly distinct branding. If you
run your own network or public service based on this code, give it its own
name, docs, and presentation.

The phrase `official Obscura47 network` refers only to the operator-run shared
network designated by the project owner.

## Questions

other than truthful reference, obtain explicit written permission first.
